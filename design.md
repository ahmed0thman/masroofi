# Design Philosophy

**Warm Tactile Minimalism**: Voice-first, tactile, and frictionless. Reduced cognitive load through physical-feeling digital spaces and Arabic-native design, now optimized for a deep, immersive **dark mode** experience.

# Principles

1. **Voice First**: The mic is always one tap away.
2. **Zero Friction**: Max 2 primary actions per screen.
3. **Privacy by Design**: On-device processing.
4. **Arabic-Native**: RTL by default with Eastern Arabic numerals.
5. **Progressive Intelligence**: System grows with the user.
6. **Feedback Everywhere**: Haptic, audio, and visual cues.
7. **Physicality**: Organic shapes, soft shadows, and tonal layering, utilizing **Plus Jakarta Sans** for a modern finish.
